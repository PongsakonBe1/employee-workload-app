## ภาพรวม

ปีการศึกษา 2569 ของ มจพ **ไม่ใช่** ปฏิทินทั่วไป (ต.ค.-ก.ย.) แต่เป็นระบบ **Fiscal Year** ที่เริ่มต้นต่างกันตามระดับการศึกษา:

| ระดับ | เริ่มภาคเรียนที่ 1 | จบภาคเรียนที่ 2 |
|-------|-------------------|----------------|
| **ปริญญาตรี + ปวช** | มิถุนายน 2569 | มีนาคม 2570 |
| **TGGS / นานาชาติ** | สิงหาคม 2569 | พฤษภาคม 2570 |

## 4. วิธีใช้ในโค้ด (สำหรับ [SE])

### Import

```javascript
import {
  ACADEMIC_PERIODS_2569_BACHELOR,
  ACADEMIC_PERIODS_2569_TGGS,
  getAcademicPeriod,
  getExamPeriods
} from "@/lib/analytics";
```

### ดึงข้อมูลช่วงเวลา

```javascript
// Default: ปริญญาตรี/ปวช
const period = getAcademicPeriod("2026-08-20");
// → { key: "sem1_mid_exam", label: "สอบกลางภาค ที่ 1", type: "peak", ... }

// TGGS/นานาชาติ
const tggsPeriod = getAcademicPeriod("2026-10-01", "tggs");
// → { key: "tggs_sem1_mid_exam", label: "สอบกลางภาค ที่ 1 (TGGS)", ... }
```

### ดึงช่วงสอบทั้งหมด

```javascript
// ช่วงสอบปริญญาตรี
const bachelorExams = getExamPeriods("bachelor");
// → [{ key: "sem1_mid_exam", ... }, { key: "sem1_final_exam", ... }, ...]

// ช่วงสอบ TGGS
const tggsExams = getExamPeriods("tggs");
```

### ใช้กับ Recharts (SeasonalPatternChart)

```javascript
// แมพสีตามประเภทช่วงเวลา
const getBarColor = (date) => {
  const period = getAcademicPeriod(date);
  return period?.color || "#94a3b8"; // default slate-400
};

// ตัวอย่าง data สำหรับ chart
const chartData = [
  { month: "2026-06", count: 45, period: getAcademicPeriod("2026-06-15") },
  { month: "2026-08", count: 120, period: getAcademicPeriod("2026-08-20") }, // peak
  { month: "2026-10", count: 150, period: getAcademicPeriod("2026-10-20") }, // peak
];
```
